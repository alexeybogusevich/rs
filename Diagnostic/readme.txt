# Installation

1. Install Python. Add it to system paths.

2. Install pip. add C:\Python34\Scripts to system paths.

3. Install Microsoft Visual C++ 10.0. Link:
http://download.microsoft.com/download/1/D/9/1D9A6C0E-FC89-43EE-9658-B9F0E3A76983/vc_web.exe

4. Install CMake. Link:
https://github.com/Kitware/CMake/releases/download/v3.14.6/cmake-3.14.6-win32-x86.msi
(Click add to All Users)
